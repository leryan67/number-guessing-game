# Number Guessing Game

A game where the user tries to guess a randomly generated number from 1-100.

## Features

-Random numbers generated each time
-Higher/lower hints
-Guess counter
-High scores

## How to Run

1. Compile the program
2. Enter guesses where prompted
3. Restart the game or exit the program
